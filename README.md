# Bank Customer Churn Analysis

## Project Overview

This project analyzes customer churn in a banking dataset using Power
BI. The dashboard highlights key churn metrics and identifies customer
segments with higher risk of leaving.

## Dashboard Highlights

-   Overall churn rate: **20.37%**
-   Total customers: **10,000**
-   Churn analysis by:
    -   Gender
    -   Age Group
    -   Geography
    -   Credit Score Range
    -   Number of Products
    -   Customer Activity Status

## Key Insights

-   Around 1 in 5 customers have churned.
-   Customers aged 50--60 have the highest churn.
-   Germany has the highest churn rate.
-   Inactive customers churn more than active customers.
-   Customers with 2 products are the most loyal.
-   Female customers show a higher churn rate than male customers.
